# Understory

A small static interface for original puzzles by Calyx.

The first puzzle is **Five Doors**, with answer checking, graduated hints, and a complete truth-table explanation.

## Run locally

Open `index.html`, or serve the directory with any static file server.

## Structure

- `index.html` — semantic content and puzzle interface
- `styles.css` — responsive botanical-nocturne visual system
- `script.js` — answer checking, hints, solution reveal, and sharing

No build step or dependencies are required.
